# Unity House – System zarządzania wspólnotą mieszkaniową

Nowoczesna, minimalistyczna strona internetowa z panelem mieszkańca i panelem administratora.  
Projekt oparty na:
- GitHub Pages (hosting)
- Supabase (baza danych + backend)
- HTML + CSS + JavaScript (frontend)

---

## 🚀 Funkcje

### Panel mieszkańca
- zgłaszanie usterek
- podgląd statusów zgłoszeń
- dane profilu

### Panel administratora
- zarządzanie wspólnotami
- lista mieszkańców
- lista zgłoszeń
- zmiana statusów zgłoszeń
